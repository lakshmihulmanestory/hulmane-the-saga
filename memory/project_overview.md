MahaHulmane is a Mahabharata-inspired fantasy web project. 28 Indian kingdoms each have a hero bonded to a sacred animal (state animal). Every kingdom also has a villain who shares the same animal but embodies its dark spirit.

**Key files:**
- `index.html` — Home page with nav, India map, collage
- `characters.html` — Lists all 28 heroes
- `character-gallery.html` — Portrait gallery with SVG art for hero + villain + animal
- `state-stories.html` — Individual kingdom story pages
- `js/characters-data.js` — All hero data (names, animals, weapons, backstories)
- `js/portrait-renderer.js` — Villain data + SVG portrait generator (added May 2026)
- `js/app.js`, `js/india-map.js` — App logic and India map
- `css/style.css` — Global styles with region color variables

**Why:** This is a creative world-building project. The user wants the story/world to feel complete, with visual character art showing how heroes, villains, and animals look.
